## Introduction
This project has been given to us to connect the three stake holder the admin who is the super user who overseas every aspect of the app, then comes enthusiasts who want to go to an adventures mountains,forests to camp,the third is the trek manager who is a trained professional who safetly conducts the trek.

## Features
Admin panel - Dashboard, Treks, Staff, Users, Bookings,Search.

Staff panel - who can see who has participated and edit the treks

The treekers -   Dashboard, Browse Treks, My Bookings, Profile

## Tech stack

- **Backend:** Flask, SQLAlchemy
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite
- **Libraries:** Flask-WTF for form handling, Jinja2 for templating
